# the-office-db

## Description

The unofficial "The Office" database project!

Project includes the following formats

- Laravel Migration and Seed (using Faker)
- JavaScript (using Faker)
- SQL Database (.sql)
- SQL Database (.dump)
- Delimited (.csv)
- JSON (.json)
